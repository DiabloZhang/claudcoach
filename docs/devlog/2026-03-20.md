## 遇到的问题
- Chrome 代理拦截本地 localhost 请求，改用 127.0.0.1 解决
- Strava Developer App 入口不在菜单，需要直接访问 strava.com/settings/api
- TablePlus 打开数据库时会锁文件，操作数据库前需关闭

## 学到的东西
- OAuth 授权流程：code 换 token，token 自动刷新
- venv 虚拟环境：每个项目独立的 Python 依赖
- SQLite 数据库被多个进程同时访问会报 "database is locked"
